// ================= НАСТРОЙКА ЗАГАДОК И ОТВЕТОВ ================= const questData = [ { riddle: "Загадка 1: Дата нашего знакомства в формате ДДММ (например, если 12 мая, то 1205)", answer: "1205" }, { riddle: "Загадка 2: Наше любимое кодовое слово или город, куда мы мечтаем отправиться вместе?", answer: "Питер" }, { riddle: "Загадка 3: Какое кодовое слово мы использовали, когда не хотели идти на пары/уроки?", answer: "эвакуация" } ]; const giftLocation = "На полке под твоим любимым свитшотом! ❤️"; // ============================================================= let currentStep = 0; let isProcessing = false; const introScreen = document.getElementById('introScreen'); const lockScreen = document.getElementById('lockScreen'); const riddleEl = document.getElementById('riddleText'); const statusEl = document.getElementById('statusBar'); const inputEl = document.getElementById('lockInputField'); const successScreen = document.getElementById('successScreen'); const giftModal = document.getElementById('giftModal'); const giftLocationEl = document.getElementById('giftLocationText'); const btnStartQuest = document.getElementById('btnStartQuest'); const btnCheckAnswer = document.getElementById('btnCheckAnswer'); const btnOpenGift = document.getElementById('btnOpenGift'); const btnCloseGift = document.getElementById('btnCloseGift'); function startQuest() { introScreen.style.display = 'none'; lockScreen.style.display = 'block'; loadStep(); } function loadStep() { riddleEl.parentElement.classList.remove('next-level-flash'); void riddleEl.offsetWidth; riddleEl.parentElement.classList.add('next-level-flash'); riddleEl.textContent = questData[currentStep].riddle; statusEl.textContent = `Уровень защиты: ${currentStep + 1} из ${questData.length}`; statusEl.style.color = '#00ff66'; setTimeout(() => { statusEl.style.color = '#555'; }, 400); inputEl.value = ""; inputEl.classList.remove('error-state'); inputEl.placeholder = "ВВЕДИТЕ ОТВЕТ..."; // Жёстко отключаем фокус и прячем клавиатуру на стыке уровней inputEl.blur(); isProcessing = false; } function checkAnswer() { if (isProcessing) return; const val = inputEl.value.trim().toLowerCase(); const ans = questData[currentStep].answer.trim().toLowerCase(); if (val === "") return; isProcessing = true; inputEl.blur(); // Скрываем мобильную клавиатуру сразу при клике на кнопку if (val === ans) { currentStep++; if (currentStep < questData.length) { loadStep(); } else { lockScreen.style.display = 'none'; successScreen.style.display = 'block'; } } else { inputEl.classList.add('error-state'); inputEl.value = ""; inputEl.placeholder = "НЕВЕРНЫЙ КОД!"; // При ошибке тоже убираем фокус, чтобы клавиатура не «прыгала» вверх-вниз inputEl.blur(); setTimeout(() => { inputEl.classList.remove('error-state'); isProcessing = false; }, 1000); } } function openGiftModal() { giftModal.style.display = 'flex'; giftLocationEl.innerHTML = "ПОДКЛЮЧЕНИЕ К ДАННЫМ...
0%.. 45%.. 89%.."; setTimeout(() => { giftLocationEl.innerHTML = `${giftLocation}`; }, 1100); } function closeGiftModal() { giftModal.style.display = 'none'; } // Привязка событий btnStartQuest.addEventListener('click', startQuest); btnCheckAnswer.addEventListener('click', checkAnswer); btnOpenGift.addEventListener('click', openGiftModal); btnCloseGift.addEventListener('click', closeGiftModal); inputEl.addEventListener('keydown', function(e) { if (e.key === 'Enter') { checkAnswer(); } }); inputEl.addEventListener('input', function() { inputEl.classList.remove('error-state'); });
